## 💻 GitHub Repositories

### 1. Algorithms & Data Structures

- **Repo:** [raywenderlich/swift-algorithm-club](https://github.com/raywenderlich/swift-algorithm-club)
- **Description:** A popular collection of data structures and algorithms implemented in Swift with detailed explanations.

### 2. Design Patterns

- **Repo:** [ochococo/Design-Patterns-In-Swift](https://github.com/ochococo/Design-Patterns-In-Swift)
- **Description:** Implementation of common Design Patterns (Singleton, Builder, Observer, etc.) in Swift.

### 3. Interview Questions

- **Repo:** [kpoltorak/Swift-Developers-Interview-Questions](https://github.com/kpoltorak/Swift-Developers-Interview-Questions)
- **Description:** Curated list of questions to test deep understanding of the language.

### 4. Style Guide

- **Repo:** [raywenderlich/swift-style-guide](https://github.com/raywenderlich/swift-style-guide)
- **Description:** The industry standard for writing clean, readable Swift code.

### 5. Awesome iOS

- **Repo:** [vsouza/awesome-ios](https://github.com/vsouza/awesome-ios)
- **Description:** A massive curated list of Swift libraries, tutorials, and frameworks.
